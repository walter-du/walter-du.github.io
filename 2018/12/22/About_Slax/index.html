关于 Slax

概述

Slax 是一个易于定制、可携带的小型 Linux 桌面发行版。
最新版 9.6.4，基于 debian 。
得益于 squashfs\autofs 以及 Slax 自身的 live 特性， 基于 Slax 可以很容易定制出针对特定用户需求的便携专用 Linux 桌面环境。
Slax 可以 burn 到 CD 或通过 USB Flash 携带，实际应用显然 USB 更普遍，以下记录也仅针对 USB 方式适用。

三种启动模式

  • Persistent changes
  • Fresh start
  • Copy to RAM

Persistent changes: 默认启动方式，从此方式启动，运行过程中针对系统的修改都会保存到 ROM (即 USB Flash );
Fresh start: 从此方式启动，运行过程中针对系统的修改保存在 RAM ，显然重启后会还原;
Copy to RAM: 从此方式启动，系统本身会 copy 到 RAM ，以加快应用启动。

实际使用时，可先通过 Persistent changes 方式安装所需的应用，将其打包成 sb ，即制作特定的 Linux 桌面环境；然后将 Fresh start 设置为默认启动方式，方便日常使用。这样可以减少对 USB Flash 的写操作消耗，毕竟好一些的 USB Flash 也就 10W+ 的写极限。

Copy to RAM 方式意义不大，当前的 USB3.0 + MLC/SLC 颗粒 USB Flash 的读速率可达 200MB+ 。

另外，从 Fresh start 启动，如果需要保存文件，可以借助 /run/initramfs/memory/data 或其它可挂载存储设备，具体使用说明参考官网。

启动后没声音

Slax 的 audio 子系统基于 alsa ;

1.确认可用声卡
#lspci -v | grep -i -A 10 audio
#dmesg | grep -i -A 10 audio
#cat /proc/asound/cards

2.查看当前声卡
#alsamixer

3.设置可用的默认声卡配置
#vi /etc/asound.conf

Add CN apt source

#vi /etc/apt/source.list

#apt update

中文显示和输入法支持

1.localtime
#ln -sf /usr/share/zoneinfo/Asia/Hong_Kong /etc/localtime

2.locale
#apt purge locales
#apt install locales
#dpkg-reconfigure locales

3.fonts
#fc-list
#apt install fonts-wqy-zenhei
#apt install fonts-wqy-microhei

#dpkg-reconfigure fontconfig-config
//#dpkg-reconfigure fontconfig

4.桌面 fluxbox 切换到 xfce
#apt install dbus-x11 task-xfce-desktop

5.scim input method
#apt install im-config zenity
#apt install scim-pinyin
//#scim -d
//#apt install fcitx-googlepinyin

Firefox 没有声音

新版本的 Firefox 需要 pulseaudio 的支持；
#apt install firefox-esr
#apt install pulseaudio
#pulseaudio -D
//#apt install pavucontrol

关于 aufs

aufs 文件系统删除文件时:
如果该文件只在 rw 目录下有，那就直接删除 rw 目录下的该文件;
如果该文件在 ro 目录下有，那么 aufs 将会在 rw 目录里面创建一个 .wh 开头的文件，标识该文件已被删除。

查看 disk I/O

#iotop -a

测试 disk I/O rate

#hdparm -Tt /dev/sda
write:
#sync; dd if=/dev/zero of=/mnt/tmp/output bs=1M count=1024; sync
read:
#dd if=/mnt/tmp/output of=/dev/null bs=1M count=1024

Login background setting

#vi /etc/lightdm/lightdm-gtk-greeter.conf
